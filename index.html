Custom Mobile Device Test

How it works

Using the addTest method from Modernizr we extend it to do a user agent test for mobile devices. We are covering around 98% of devices using the following regular expression:

/Mobile|iP(hone|od|ad)|Android|BlackBerry|IEMobile|Kindle|NetFront|Silk-Accelerated|(hpw|web)OS|Fennec|Minimo|Opera M(obi|ini)|Blazer|Dolfin|Dolphin|Skyfire|Zune/

This includes barely significant outliers and third party browsers. Anything that doesn't make this cut probably can't render any website made past 2004 anyway.
